# Spec 2：统一日历 — 多计划聚合展示

## Why

用户同时推进多个学习目标时，不应在多个独立日历间切换，而应在同一日历中看到所有计划的任务安排。

## What Changes

- 实现 UnifiedCalendar 月历组件
- 实现多计划按日期聚合逻辑
- 实现每日预览（Modal/Drawer）
- 支持上/下月切换

## Requirements

1. 日历展示**所有已保存计划**的任务，按日期**合并**（不是单个计划的日历）
2. 月历视图，按周排列日期格子
3. 支持切换上一个月和下一个月
4. 有学习任务的日期需要高亮
5. 日期格子显示当天总学习时长或任务数量
6. 同一天多个计划的任务合并显示在一个格子中
7. 点击没有任务的日期：不弹出错误，无操作或轻提示
8. 点击有任务的日期：打开每日预览

## Scenarios

| 场景 | 预期结果 |
|------|----------|
| 用户有「React 基础」和「线性代数复习」两个计划，6月2日都有任务 | 6月2日只出现一个格子，显示合计时长（如 3h） |
| 用户切换到上个月/下个月 | 日历正确显示对应月份，有任务日期仍高亮 |
| 用户点击无任务日期 | 不报错，不打开预览 |
| 用户点击有任务日期 | 打开每日预览，显示当天所有计划任务 |
| 用户删除一个计划 | 日历同步更新，被删计划日期不再高亮（若无其他计划任务） |
